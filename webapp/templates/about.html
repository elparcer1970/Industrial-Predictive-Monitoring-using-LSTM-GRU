About This Project

A production-quality deep learning system for industrial predictive maintenance, built on NASA's turbofan engine degradation dataset.

01 What It Does

Given a window of {{ info.window }} historical sensor readings, the system produces three simultaneous outputs from a single neural network inference:

  • Sensor Forecast — predicted next-timestep value for each of the {{ info.n_sensors }} sensor channels.
  • Remaining Useful Life (RUL) — how many engine cycles remain before predicted failure. Max is {{ info.max_rul }} cycles.
  • Anomaly Score — reconstruction error between prediction and actual reading. Above threshold ({{ info.threshold }}) triggers an alert.

03 Feature Engineering Pipeline

1
Z-score Normalisation

Per-column standardisation at inference time

2
Rolling Statistics

Window=10: mean, std, trend (diff) per sensor

3
Cross-sensor Ratios

Pairwise ratios between first 3 sensor pairs

4
MAD Health Score

Mean absolute deviation across all sensors

5
Sliding Window

Last 30 timesteps → shape (1, 30, {{ info.n_features }})

This pipeline expands {{ info.n_sensors }} raw sensors to {{ info.n_features }} engineered features, giving the model rich temporal context.
